# Claude Project Instructions

Tuân thủ [AGENTS.md](AGENTS.md) và skill tại `.claude/skills/coding-hcm1-app/SKILL.md`.

Ưu tiên:

1. Bảo vệ dữ liệu giáo viên.
2. Giữ giao diện sáng và dễ đọc.
3. Chạy `npm run lint` và `npm run build` trước khi bàn giao.
4. Không ghi bí mật vào mã nguồn hoặc tài liệu.
